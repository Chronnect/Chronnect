API Information

Use the following requests to retrieve information linked to an ETH address, including contact details and the associated PGP Public Key. Please note that in the examples provided, the ETH address is passed to the interface without the leading '0x'.

Fetch Contact Info by ETH Address

curl -X POST "https://node0.projectnet.chromia.dev:7740/query/A96C12971ED1C2036C762A99909121CDD39C62B000BEAE982B529CFE6F61398F" \
	-H "accept: application/json" \
	-H "content-type: application/json" \
	-d '{"type":"fetch_contact_info_by_eth","eth_address":"9657d0E9B4F2A4031a79bC603Ca3A1b9cd46fAc9"}'

Fetch PGP Public Key by ETH Address

curl -X POST "https://node0.projectnet.chromia.dev:7740/query/A96C12971ED1C2036C762A99909121CDD39C62B000BEAE982B529CFE6F61398F" \
	-H "accept: application/json" \
	-H "content-type: application/json" \
	-d '{"type":"fetch_pgp_public_key_by_eth","eth_address":"9657d0E9B4F2A4031a79bC603Ca3A1b9cd46fAc9"}'
